## ğŸƒ How to Setup

- Download or clone the repository
- Move the project to the selected directory
- Find the database [hungryme_desktop/database/hungryme_desktop.sql](./database/hungryme_desktop.sql)
- Create a database named `hungryme` in your localhost (phpMyAdmin)
- Import the `hungryme_desktop.sql` file to the `hungryme` database
- Open project [hungryme_desktop.csproj](./hungryme_desktop.csproj) or solution [hungryme_desktop.sln](./hungryme_desktop.sln) with Visual Studio
- Clean the solution (or project) and rebuild
